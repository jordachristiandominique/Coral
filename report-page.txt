What Each Report Type Contains:
Summary Report
Sections:

Cover page with logo and metadata
Executive summary (1 page)
Overview statistics:

Total surveys conducted
Total images analyzed
Average coverage percentage
Biodiversity distribution (pie chart)


Geographic distribution map
Temporal trends (line chart)
Key findings (3-5 bullet points)
Methodology brief
Sample images (3-6 representative photos)
Recommendations

Page count: 8-12 pages

2. Batch-Specific Report
Sections:

Batch metadata (name, date, location, researcher)
Survey details (quadrat size, conditions, notes)
Analysis results:

Coverage percentage
Biodiversity class
Point-by-point breakdown (if using 10-point method)


All images with individual results
Location map with single marker
Comparison to previous surveys at same location (if available)
Researcher observations
Raw data appendix

Page count: 5-8 pages

3. Trend Analysis Report
Sections:

Time period overview
Coverage trend chart (line graph over time)
Biodiversity class changes (stacked area chart)
Month-by-month or quarter-by-quarter breakdown
Statistical analysis (mean, median, std deviation)
Correlation analysis (if multiple locations)
Interpretation of trends
Predictions/forecasts (optional)
Recommendations based on trends

Page count: 10-15 pages

4. Location Comparison Report
Sections:

Side-by-side location profiles
Comparison table (coverage, class, etc.)
Comparative charts (grouped bar charts)
Map showing all locations
Statistical significance tests (optional)
Best/worst performing sites
Factors analysis (why differences exist)
Site-specific recommendations

Page count: 12-20 pages (depends on # of locations)

5. Biodiversity Distribution Report
Sections:

Overview of biodiversity classification system
Class A/B/C distribution (pie charts, maps)
Geographic patterns (spatial analysis)
Temporal patterns (how distribution changed)
Hotspot identification (where are healthy reefs?)
Degradation zones (where intervention needed)
Conservation priority ranking
Actionable recommendations

Page count: 10-15 pages
